# Expense Tracker (under development)

Simple desktop app to track personal income & expenses.

**Tech**: Python • Tkinter • CSV • Matplotlib (optional charts)

## Features
- Add / edit / delete transactions (income & expense)
- Predefined categories with color support
- Monthly budgets per category + total limit
- Budget overage alerts
- Filter by month / category / keyword
- Monthly summary (totals + per-category breakdown)
- Spending pie chart
- Data saved to `expenses.csv`
